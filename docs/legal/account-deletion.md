# AppTag 계정 삭제 안내

AppTag 계정을 삭제하시려면 아래 두 가지 방법 중 하나를 선택하세요. 두 방법 모두 동일하게 **모든 데이터가 즉시 삭제**됩니다.

---

## 방법 1 — 앱 내에서 직접 삭제 (권장)

1. AppTag 앱 실행
2. 하단 **"프로필"** 탭 이동
3. **"계정 삭제"** 버튼 탭
4. 확인 다이얼로그에서 **"삭제"** 선택

→ 즉시 모든 데이터가 삭제되고 계정이 폐기됩니다.

## 방법 2 — 이메일로 삭제 요청

앱이 설치되어 있지 않거나 앱 내 삭제가 곤란한 경우:

- **수신 이메일**: tawooltag@gmail.com
- **제목**: "AppTag 계정 삭제 요청"
- **본문에 포함할 내용**:
  - 가입한 이메일 주소
  - 가입 방법 (이메일 / Google / Apple)

**처리 기한**: 영업일 기준 **5일 이내** 처리되며, 처리 완료 시 확인 이메일을 발송합니다.

---

## 삭제되는 데이터

| 항목 | 처리 |
|------|------|
| 계정 정보 (이메일, 프로필) | 즉시 삭제 |
| 클라우드 백업 템플릿 | 즉시 삭제 |
| 로컬 기기 데이터 (Hive) | 앱 삭제 시 자동 제거 |
| 에러·크래시 로그 | 90일 이내 자동 삭제 |

## 보관되는 데이터

**없음.** 법령에 따른 별도 보관 의무가 없으므로 모든 데이터가 삭제 처리됩니다.

## Account Deletion (English)

You can delete your AppTag account in either of the following ways. Both immediately delete all your data.

### Option 1 — In-App Deletion (Recommended)
1. Open AppTag
2. Go to the **"Profile"** tab
3. Tap **"Delete Account"**
4. Confirm in the dialog

### Option 2 — Email Request
If you cannot access the app:
- Email: **tawooltag@gmail.com**
- Subject: "AppTag Account Deletion Request"
- Include: Your signup email, sign-in method (Email / Google / Apple)

Processed within **5 business days**.

### Deleted Data
- Account info (email, profile)
- Cloud backup templates
- Local device data (upon app uninstall)
- Error logs (auto-deleted within 90 days)

### Retained Data
**None.** All data is deleted.
