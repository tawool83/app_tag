# AppTag 개인정보처리방침

**최종 업데이트**: 2026-04-23
**시행일**: 2026-04-23

AppTag(이하 "본 앱")는 사용자의 개인정보를 소중히 여기며, 「개인정보 보호법」 등 관련 법령을 준수합니다. 본 방침은 본 앱이 처리하는 개인정보의 항목·목적·보관·파기·사용자 권리에 대해 안내합니다.

---

## 1. 수집하는 개인정보 항목

### 1.1 회원가입·로그인 시 (선택)
본 앱의 주요 기능은 비로그인으로 사용 가능합니다. 클라우드 백업 등 일부 기능 사용 시에만 다음 정보를 수집합니다.

| 항목 | 수집 시점 | 수집 방법 |
|------|----------|----------|
| 이메일 주소 | 이메일 회원가입 시 | 사용자 직접 입력 |
| 비밀번호 (해시값) | 이메일 회원가입 시 | 사용자 직접 입력 (Supabase 가 해시 처리) |
| 이름·이메일·프로필 사진 | Google·Apple 로그인 시 | 해당 SSO 제공자가 전달 |

### 1.2 서비스 이용 시 자동 수집
- 기기 정보 (OS 버전, 앱 버전, 단말 모델)
- 에러·크래시 로그 (서비스 안정성 진단 목적)

### 1.3 사용자가 직접 입력·생성한 데이터
- QR 코드 콘텐츠 (URL, 텍스트, 연락처, 위치 좌표, Wi-Fi 정보 등)
- 사용자 정의 템플릿·스티커 텍스트
- 스캔 히스토리 (사용자 기기 내부에만 저장)

## 2. 개인정보 수집·이용 목적

| 목적 | 사용 항목 |
|------|----------|
| 회원 식별 및 계정 관리 | 이메일, SSO 제공 정보 |
| 사용자 생성 QR 코드의 클라우드 백업 (선택) | 사용자 콘텐츠, 계정 식별자 |
| 서비스 안정성 및 품질 개선 | 기기 정보, 에러 로그 |
| 고객 문의 응대 | 이메일 |

---

## 3. 보관 기간 및 파기

| 항목 | 보관 기간 |
|------|----------|
| 회원 계정 정보 | 회원 탈퇴 시 즉시 삭제 |
| 클라우드 백업 데이터 | 회원 탈퇴 시 즉시 삭제 |
| 로컬 기기 저장 데이터 | 앱 삭제 시 자동 제거 |
| 에러·크래시 로그 | 90일 자동 삭제 |

법령에 따른 별도 보관 의무는 없습니다.

---

## 4. 제3자 제공 및 처리 위탁

본 앱은 사용자의 개인정보를 제3자에게 제공하지 않습니다. 다만 서비스 운영을 위해 다음 외부 서비스를 이용합니다.

| 위탁 업체 | 위탁 업무 | 데이터 보관 위치 |
|-----------|----------|-----------------|
| Supabase, Inc. | 회원 인증, 데이터베이스 호스팅 | ap-northeast-2 (서울) |
| Google LLC | Google 계정 로그인 (SSO) | Google 정책 준수 |
| Apple Inc. | Apple 계정 로그인 (Sign in with Apple) | Apple 정책 준수 |

## 5. 사용자의 권리

사용자는 언제든지 다음 권리를 행사할 수 있습니다:
- 개인정보 **열람** 요청
- 개인정보 **정정** 요청
- 개인정보 **삭제** 요청 (계정 삭제)
- 개인정보 처리 **정지** 요청

1. **앱 내**: "프로필 > 계정 삭제" 메뉴
2. **이메일**: tawooltag@gmail.com (영업일 기준 5일 이내 처리)

## 7. 권한 사용 안내

본 앱은 다음 단말 권한을 사용합니다. 모든 권한은 사용자가 명시적으로 해당 기능을 사용할 때에만 요청됩니다.

| 권한 | 사용 목적 |
|------|----------|
| 카메라 | QR 코드 스캔 |
| NFC | NFC 태그 읽기·쓰기 |
| 위치 | 지도 태그 생성 시 현재 위치 |
| 연락처 | 연락처 QR 생성 |
| 사진/미디어 | 로고 이미지 선택, QR 이미지 갤러리 저장 |
| 앱 목록 조회 | 설치된 앱 딥링크를 QR 로 저장하기 위한 앱 선택 |
